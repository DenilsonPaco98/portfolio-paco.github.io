1️⃣ CI/CD AUTOMÁTICO (O que você pediu)
═══════════════════════════════════════════════════════════════════════════════

  GitHub Actions: .github/workflows/deploy.yml
  
  Fluxo:
  ┌─────────────────────────────────────────────────────────┐
  │                   Você trabalha local                    │
  │  git add . → git commit → git push origin develop       │
  └──────────────────┬──────────────────────────────────────┘
                     │
                     ▼
  ┌─────────────────────────────────────────────────────────┐
  │          GitHub Actions Triggered Automático             │
  │  • Checkout code                                        │
  │  • Setup Node.js 20                                     │
  │  • Install dependencies                                 │
  │  • Lint HTML                                            │
  │  • Deploy to GitHub Pages (se main branch)              │
  └──────────────────┬──────────────────────────────────────┘
                     │
                     ▼ (1-2 minutos)
  ┌─────────────────────────────────────────────────────────┐
  │     ✅ Site atualizado em https://paco.dev               │
  └─────────────────────────────────────────────────────────┘

  Ambientes:
  • main (produção) → Deploy automático
  • develop (staging) → Validação apenas
  
  Você nunca mais precisa fazer deploy manual! 🚀
