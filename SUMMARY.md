💡 DICAS IMPORTANTES
═══════════════════════════════════════════════════════════════════════════════

1. PRIMEIRO DEPLOY
   • Sempre faça PR antes de merge para main
   • Veja logs em GitHub Actions
   • Espere 1-2 min para deploy completar
   • Limpe cache do navegador (Ctrl+Shift+Del)

2. ATUALIZAR ROADMAP
   • Edite % diretamente no index.html
   • Procure por "roadmap-badge" e "progress-fill"
   • Mude a classe do badge (completed/in-progress/upcoming)
   • Push automático faz deploy

3. ADICIONAR CERTIFICAÇÃO
   • Copie um card existente
   • Mude: nível, nome, código
   • Ajuste tópicos e timeline
   • Defina status e % inicial
   • Push e vê no site em 2 min!

4. TROUBLESHOOTING
   • Deploy não funciona? Veja Actions no GitHub
   • Mudanças não aparecem? Limpe cache
   • Erro de sintaxe? Validate HTML
   • Confira /docs/CI-CD.md section Troubleshooting
