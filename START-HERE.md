💡 DICAS IMPORTANTES
═══════════════════════════════════════════════════════════════════════════════

1. PRIMEIRO DEPLOY
   • Sempre use branches (develop para features)
   • Abra PR antes de merge
   • Veja logs em GitHub Actions
   • Espere 1-2 min para deploy

2. ATUALIZAR CERTIFICAÇÕES
   • Edite % no index.html
   • Mude classe do badge
   • git push automático faz deploy
   • Site atualiza em 2 minutos

3. ADICIONAR CERTS
   • Copie um card existente
   • Customize: nome, código, tópicos
   • Defina status e % inicial
   • git push

4. TROUBLESHOOTING
   • Mudanças não aparecem? Limpe cache (Ctrl+Shift+Del)
   • Deploy falhou? Veja Actions → Logs
   • Erro de sintaxe? Validate HTML
   • Confira /docs/CI-CD.md seção Troubleshooting
